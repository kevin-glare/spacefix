# Spacefix
Simple gem designed to prevent extra whitespace from sneaking into string field.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'spacefix'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install spacefix
